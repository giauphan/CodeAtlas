# CodeAtlas MCP Server

A standalone [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server that exposes [CodeAtlas](https://github.com/giauphan/CodeAtlas) code analysis data to AI assistants.

## Overview

This MCP server reads `.codeatlas/analysis.json` files generated by the [CodeAtlas VS Code extension](https://marketplace.visualstudio.com/items?itemName=giauphan.codeatlas) and provides tools for querying:

- **Project structure** — modules, classes, functions, variables
- **Dependencies** — import/call/containment relationships
- **AI insights** — refactoring suggestions, security issues, maintainability
- **Entity search** — fuzzy search across all entities
- **File entities** — all entities defined in a specific file

## Prerequisites

- [Node.js](https://nodejs.org/) 18+
- [CodeAtlas VS Code extension](https://marketplace.visualstudio.com/items?itemName=giauphan.codeatlas) installed in your editor
- At least one project analyzed (run `CodeAtlas: Analyze Project` in VS Code)

## Installation

```bash
git clone https://github.com/giauphan/codeatlas-mcp.git
cd codeatlas-mcp
npm install
```

## Usage

### Start the server

```bash
npm start
```

The server runs on **stdio** and auto-discovers all projects in your home directory that have `.codeatlas/analysis.json`.

### Configure with AI Assistants

Add to your MCP configuration (e.g. `mcp_config.json`):

```json
{
  "mcpServers": {
    "codeatlas": {
      "command": "npx",
      "args": ["tsx", "/path/to/codeatlas-mcp/index.ts"]
    }
  }
}
```

### Environment Variables

| Variable | Description |
|---|---|
| `CODEATLAS_PROJECT_DIR` | Optional. Specify a project directory to prioritize |

## Available Tools

| Tool | Description |
|---|---|
| `list_projects` | List all discovered analyzed projects |
| `get_project_structure` | Get modules, classes, functions, variables |
| `get_dependencies` | Get import/call/containment relationships |
| `get_insights` | Get AI-generated code insights |
| `search_entities` | Search entities by name (fuzzy match) |
| `get_file_entities` | Get all entities in a specific file |

## How It Works

1. The [CodeAtlas extension](https://github.com/giauphan/CodeAtlas) analyzes your project and saves results to `.codeatlas/analysis.json`
2. This MCP server discovers and reads those analysis files
3. AI assistants query the server via MCP protocol to understand your codebase
